video 태그 속성

controls : 동영상 재생 컨트롤러 표시 여부 지정
autoplay : 페이지 로딩 시 동영상 자동 재생 여부 지정
loop : 동영상 재생이 끝나면 자동으로 처음부터 재생 여부 지정
muted : 동영상 소리 음소거 여부 지정
poster : 동영상 로드 되기 전 표시되는 이미지 지정
width : 동영상 너비 지정
height : 동영상 높이 지정

<video width="" height="" controls>
    <source src="" type="video/mp4"
    <source src="" type="video/ogg"
    동영상을 불러올 수 없습니다.
</video>

}
video::-ms-media-controls {
    /*IE*/
    color: red;

</style>
